Aufgabe 1: Ein kleines CSS-Quiz

Welcher CSS-Selektor wählt …

… alle Absätze der Seite aus?=

  1. div
  2. p.absatz
  3. p > .absatz
  4. p

… alle Elemente der Klasse „info” aus?

  1. info
  2. #info
  3. .info
  4. info()

… das Element mit der ID „page-title” aus?

  1. page-title
  2. #page-title
  3. .page-title
  4. page #title

… alle Elemente der Klasse „active” aus, die sich innerhalb eines <nav>-Elements befinden?

  1. nav .active
  2. nav > .active
  3. nav.active
  4. nav #active

Mit welcher CSS-Anweisung kann man …

… die Schriftart festlegen?

  1. font-size
  2. font-family
  3. font-weight
  4. text-decoration

… ein Hintergrundbild setzen?

  1. background-color
  2. background-image
  3. background-size
  4. background

… den Innenabstand vom Rahmen eines Elements zu seinem Inhalt definieren?

  1. padding
  2. margin
  3. border

… einen Schatten definieren?

  1. border-radius
  2. text-decoration
  3. text-shadow
  4. box-shadow

Lösung: 4, 3, 2, 1 und 2, 2, 2 und 4, 1, 3 und 4

Hoffentlich hast du nicht schon wieder alles vergessen. 😁 Dieses kleine Quiz soll dir helfen, das zu prüfen. 📑

Bildnachweis: Atom Webseite

Spaß mit Hintergrundbildern

  • Gekachelter Hintergrund
  • Ein großes Hintergrundbild
  • Parallax-Effekt

Beispiel ansehen

... Vor allem in den frühen Jahren des Web hat man diesen Effekt häufig gesehen. Heute wirkt es jedoch oft altbacken und wird daher nicht mehr so angewendet.

GEKACHELTER HINTERGRUND

Beispiel ansehen

... Dieser Effekt war eine Zeit lang modern und kommt immer mal wieder zum Einsatz, wenn ein einfarbiger Hintergrund zu langweilig aussieht. Allerdings darf das Hintergrundbild nicht zu unruhig wirken und muss einen guten Kontrast zur Schrift ergeben, entweder weil das Bild nur wenige Farben besitzt oder weil du den Text in kleine Kästchen mit einfarbigem Hintergrund setzt.

EIN GROSSES HINTERGRUNDBILD

Beispiel ansehen

... Du hast sicher auch schon immer mal wieder Webseiten gesehen, bei der sich die ganze Musik auf einer einzigen, langen Seite abspielt. Gerade bei Produktpräsentationen ist dies ganz beliebt. Damit die Seite nicht so statisch wirkt, wird dabei oft ein sogenannter Parallax-Scrolling-Effekt eingesetzt, bei dem der Hintergrund anders scrollt als der Vordergrund. Manche Webdesigner verkünsteln sich hier regelrecht, das grundlegende Layout ist aber einfach zu erstellen.

PARALLAX-EFFEKT
...

Aber denke daran, dass zu viele Hintergrundbilder die Ladezeit deiner Seite erhöhen. Du solltest es daher gerade mit dem Parallax-Effekt nicht übertreiben und wie in den HTML-Folien gezeigt, die Dateigröße der Bilder so gut wie möglich optimieren. 🤓
